# COMPATECH
COMPATECH
Projeto desenvolvido para a disciplina de Desenvolvimento Web.
Descrição:
O COMPATECH é um site voltado para comparação de produtos de tecnologia, incluindo processadores, placas de vídeo, televisores e celulares.

Tecnologias utilizadas
* HTML5
* CSS3
* JavaScript
* json

Funcionalidades

* Comparação de produtos
* Navegação entre categorias
* Interface responsiva
